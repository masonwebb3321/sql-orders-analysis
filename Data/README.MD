# SQL Orders Analysis
This project demonstrates SQL-based analysis of customer ordering behavior
using a relational SQLite database and a Jupyter notebook.

## Overview
The analysis explores how users interact with an ordering system by examining
order frequency, revenue, product popularity, and repeat behavior.

## Data
- SQLite database with Users, Orders, and Order_Items tables
- Raw data excluded or anonymized for privacy

## Key Concepts Demonstrated
- Multi-table joins
- Aggregations and grouping
- Window functions
- Business metric design

## Analysis
Notebook:  /notebooks/orders_sql_analysis.ipynb

## Tools Used
- SQL (SQLite)
- Python (Jupyter, pandas)
